# FleetNow Parcel Size & Weight Policy (2026)

## 1. Size Classifications

If an order has multiple packages, classify by **TOTAL combined volume**, not each one individually. If the package meets different tiers on different dimensions, charge based on the **highest tier**.

| Tier | Each side ≤ | Weight ≤ | Surcharge |
|---|---|---|---|
| Standard | 43×31×31 cm | 15 lb (6.8 kg) | $0 |
| Medium | 60×31×31 cm | 30 lb (13.6 kg) | +$4 + tax |
| Large | 80×38×38 cm | 50 lb (22.7 kg) | +$9 + tax |

## 2. Multi-Package Orders

The size tier is determined by the **TOTAL COMBINED volume** of all packages, not each one.

Example: 3 packages of 40×20×20 cm — combined volume is larger than Standard. Check whether combined dimensions fit Medium or Large limits.

## 3. Prohibited / Oversized (cannot ship)

CANNOT ship if ANY of the following is true:
- Any single side > 80 cm
- Total weight > 50 lb (22.7 kg)

For oversized/overweight items, direct the customer to call **+1 416-649-6588** for pallet/freight options. Do NOT quote a price for prohibited items.

## 4. Decision Rules (use this for size questions)

1. If dimensions/weight not provided, ASK: "What are the approximate dimensions (L × W × H) and weight of your package(s)?"
2. Determine the tier (use total combined volume for multi-package).
3. Total = Region base + size surcharge if Medium / Large.
4. Be transparent when quoting: "Your package is classified as [Medium/Large], which includes a $[4/9] size surcharge."
5. If exceeds all limits: "I'm sorry, this package exceeds our standard size limits. Please contact us at +1 416-649-6588 to discuss freight/pallet options."
